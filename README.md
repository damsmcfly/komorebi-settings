My Komorebi commands cheat sheet:


# Start Komorebi & WHKD : 
```
  komorebic start --whkd 
```

# Stop Komorebi & WHKD : 
```
komorebic stop
```
or
```
  Stop-Process -Name whkd -Force -ErrorAction SilentlyContinue  
  Stop-Process -Name komorebi -Force -ErrorAction SilentlyContinue
```
